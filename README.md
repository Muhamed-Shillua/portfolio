# Muhamed Shillua – Portfolio

## Overview

This is a personal portfolio website showcasing my skills as a Backend Developer with Node.js expertise and frontend experience. The project demonstrates clean modular architecture, modern JavaScript patterns, and responsive UI design.

## Features

* **Frontend**

  * Modular JS components (`hero.js`, `skills.js`, etc.)
  * Smooth UI interactions using `IntersectionObserver` and `requestAnimationFrame`
  * Dark/Light theme toggle with CSS variables
  * Responsive design (works on desktop & mobile)
* **Backend & API**

  * Project structure inspired by scalable backend patterns
  * Ready for dynamic content integration
* **Security & Best Practices**

  * Safe external links with `rel="noopener noreferrer"`
  * SEO-friendly meta tags for social previews
  * Minimal dependencies, fully vanilla JS
